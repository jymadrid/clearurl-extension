## ✨ 功能特性

### 核心功能

- **🔒 自动净化URL**：使用 declarativeNetRequest API 实时移除跟踪参数
- **⚡ 零延迟处理**：浏览器级别处理确保即时净化，无性能影响
- **🎯 智能检测**：识别并移除 60+ 种跟踪参数（UTM、Facebook、Google Analytics 等）

### 高级功能

- **📋 剪贴板自动净化**：复制链接时自动净化，确保分享的链接始终干净
- **🔗 短链接预解析**：自动展开并净化短链接（t.co、bit.ly 等），直达干净目标
- **🎨 自定义白名单**：添加信任的网站，不对其 URL 进行净化
- **⚙️ 自定义规则**：定义您自己的 URL 参数移除规则
- **📊 净化日志**：可视化展示所有净化记录，包含详细信息
- **📈 统计面板**：追踪已净化的 URL 和参数数量

### 用户体验

- **🌐 多标签界面**：易用的弹出窗口，包含概览、白名单、自定义规则、净化日志等标签页
- **🔔 桌面通知**：链接自动净化时收到通知提醒
- **💾 数据持久化**：所有设置和日志本地保存
- **🎛️ 精细控制**：根据需求单独启用/禁用各项功能

## 📦 安装指南

### **官方商店 (即将发布)**
片刻等待，即可在 Chrome 网上应用店一键安装，享受官方带来的便捷与安全。

### **先锋体验 (开发者模式)**
如果您迫不及待，希望立即体验，可以从源代码进行安装：

1.  **克隆仓库**
    ```bash
    git clone https://github.com/jymadrid/clearurl-extension.git
    ```
2.  **进入 Chrome 扩展管理页面**
    在地址栏输入 `chrome://extensions/` 并回车。

3.  **开启开发者模式**
    激活页面右上角的"开发者模式"开关。

4.  **加载扩展**
    点击"加载已解压的扩展程序"，然后选中您刚刚克隆的 `clearurl-extension` 文件夹。

完成。ClearURL 图标将静静地出现在您的浏览器工具栏，开始它的守护使命。

## 🛠️ 工作原理

ClearURL 的简约背后，是强大技术的支撑。

### 核心技术

- **Manifest V3**: 我们采用 Google 最新的扩展规范，从架构上保证了更高的安全性、稳定性和性能。
- **Service Worker (`background.js`)**: 作为扩展的智能中枢，它在独立的后台线程中运行，负责逻辑处理与状态管理，确保主线程的流畅。
- **Declarative Net Request**: 这是实现"零延迟"体验的核心技术。我们通过预设的声明式规则，将 URL 净化这一繁重任务，完全交由浏览器内核高效处理，彻底告别了传统 JavaScript 阻断方式带来的性能瓶颈。

### 高级机制

- **剪贴板监控**：使用 `chrome.alarms` API 定期检查剪贴板内容（每2秒），自动净化 URL
- **短链接解析**：采用 `fetch HEAD` 请求解析重定向，不下载页面内容，支持最多5次重定向跳转
- **动态规则更新**：修改白名单或自定义规则时自动更新 declarativeNetRequest 规则
- **标签页更新追踪**：监控标签页 URL 变化，实时记录净化活动
